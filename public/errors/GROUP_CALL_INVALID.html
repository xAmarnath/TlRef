GROUP_CALL_INVALID

Group call invalid.

HTTP 400 Calls / Voice / Video

When & why it happens

The supplied group call is malformed, points to nothing the server recognises, or fails a server-side validity check.

Handling in gogram

Surfaced as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "GROUP_CALL_INVALID" { // handle this error } }
