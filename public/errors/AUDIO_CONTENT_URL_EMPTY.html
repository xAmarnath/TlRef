HTTP 400 · Bad Request

AUDIO_CONTENT_URL_EMPTY

The remote URL specified in the content field is empty.

HTTP 400 Media / Files

When & why it happens

A required audio content url was omitted from the request. Populate it and retry.

Handling in gogram

Surfaced as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "AUDIO_CONTENT_URL_EMPTY" { // handle this error } }
