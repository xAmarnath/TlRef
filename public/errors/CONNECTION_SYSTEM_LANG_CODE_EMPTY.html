CONNECTION_SYSTEM_LANG_CODE_EMPTY

The system language string was empty during connection.

When & why it happens

A required connection system lang code was omitted from the request. Populate it and retry.

Handling in gogram

Gogram surfaces this as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "CONNECTION_SYSTEM_LANG_CODE_EMPTY" { // handle this error } }
