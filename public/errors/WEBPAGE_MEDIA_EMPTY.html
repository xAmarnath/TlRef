WEBPAGE_MEDIA_EMPTY

Webpage media empty.

HTTP 400 Web previews

When & why it happens

A required webpage media was omitted from the request. Populate it and retry.

Handling in gogram

Surfaced as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "WEBPAGE_MEDIA_EMPTY" { // handle this error } }
