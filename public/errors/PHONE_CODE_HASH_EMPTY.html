PHONE_CODE_HASH_EMPTY

phone_code_hash is missing.

When & why it happens

A required phone code hash was omitted from the request. Populate it and retry.

Handling in gogram

Gogram surfaces this as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "PHONE_CODE_HASH_EMPTY" { // handle this error } }
