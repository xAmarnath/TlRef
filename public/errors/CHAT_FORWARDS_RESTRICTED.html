CHAT_FORWARDS_RESTRICTED

You can't forward messages from a protected chat.

HTTP 403 Permissions / Forbidden

When & why it happens

You can't forward messages from a protected chat. Treated by gogram as a regular API error; inspect ErrResponseCode.Message for the exact code.

Handling in gogram

Surfaced as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "CHAT_FORWARDS_RESTRICTED" { // handle this error } }
