PUBLIC_KEY_REQUIRED

A public key is required.

HTTP 400 Other

When & why it happens

A public key is required. Treated by gogram as a regular API error; inspect ErrResponseCode.Message for the exact code.

Handling in gogram

Surfaced as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "PUBLIC_KEY_REQUIRED" { // handle this error } }
