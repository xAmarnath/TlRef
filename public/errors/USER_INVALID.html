USER_INVALID

Invalid user provided.

When & why it happens

The supplied user is malformed, points to nothing the server recognises, or fails a server-side validity check.

Handling in gogram

Gogram surfaces this as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "USER_INVALID" { // handle this error } }
