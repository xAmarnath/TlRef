HTTP 400 · Bad Request

CHAT_PUBLIC_REQUIRED

You can only enable join requests in public groups.

HTTP 400 Channels & Chats

When & why it happens

You can only enable join requests in public groups. Treated by gogram as a regular API error; inspect ErrResponseCode.Message for the exact code.

Handling in gogram

Surfaced as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "CHAT_PUBLIC_REQUIRED" { // handle this error } }
