PRIVACY_KEY_INVALID

The privacy key is invalid.

When & why it happens

The supplied privacy key is malformed, points to nothing the server recognises, or fails a server-side validity check.

Handling in gogram

Gogram surfaces this as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "PRIVACY_KEY_INVALID" { // handle this error } }
