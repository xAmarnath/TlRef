HTTP 400 · Bad Request

BOOST_PEER_INVALID

The specified boost_peer is invalid.

HTTP 400 Boost / Levels

When & why it happens

The supplied boost peer is malformed, points to nothing the server recognises, or fails a server-side validity check.

Handling in gogram

Surfaced as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "BOOST_PEER_INVALID" { // handle this error } }
