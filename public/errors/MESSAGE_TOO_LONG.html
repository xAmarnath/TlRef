MESSAGE_TOO_LONG

The provided message is too long.

When & why it happens

The message text exceeds the per-message byte limit. Split it into multiple messages.

Handling in gogram

Gogram surfaces this as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "MESSAGE_TOO_LONG" { // handle this error } }
