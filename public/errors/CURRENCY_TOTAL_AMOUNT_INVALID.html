CURRENCY_TOTAL_AMOUNT_INVALID

The total amount of all prices is invalid.

When & why it happens

The supplied currency total amount is malformed, points to nothing the server recognises, or fails a server-side validity check.

Handling in gogram

Gogram surfaces this as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "CURRENCY_TOTAL_AMOUNT_INVALID" { // handle this error } }
