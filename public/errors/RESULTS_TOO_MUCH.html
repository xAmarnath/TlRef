RESULTS_TOO_MUCH

Too many results were provided.

HTTP 400 Flood / Rate limits

When & why it happens

The account already has the maximum number of results allowed. Remove some before adding more.

Handling in gogram

Surfaced as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "RESULTS_TOO_MUCH" { // handle this error } }
