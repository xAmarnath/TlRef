Polls & Quizzes HTTP 400 · Bad Request

POLL_VOTE_REQUIRED

Cast a vote in the poll before calling this method.

When & why it happens

Poll/quiz state precondition failed — usually the poll is closed, the choice is malformed, or the user already voted.

Handling in gogram

Gogram surfaces this as *gogram.ErrResponseCode. Match on the message:

if err, ok := err.(*gogram.ErrResponseCode); ok { if err.Message == "POLL_VOTE_REQUIRED" { // handle this error } }
