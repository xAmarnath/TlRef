MessagesMessages

Abstract type representing one of 4 possible constructors.

Type

Available Constructors

In Gogram, this type is represented as tg.MessagesMessages interface. Use any of the following constructors:

MessagesChannelMessages Channel messages MessagesMessages Full list of messages with auxiliary data. MessagesMessagesNotModified No new messages matching the query were found MessagesMessagesSlice Incomplete list of messages and auxiliary data.

Gogram Example

// messages.Messages is an interface type
// You can use any of the following constructors:
var _ tg.MessagesMessages = &tg.MessagesChannelMessages{}
var _ tg.MessagesMessages = &tg.MessagesMessages{}
var _ tg.MessagesMessages = &tg.MessagesMessagesNotModified{}
var _ tg.MessagesMessages = &tg.MessagesMessagesSlice{}
