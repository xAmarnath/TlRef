InputFileLocation

Abstract type representing one of 10 possible constructors.

Type

Available Constructors

In Gogram, this type is represented as tg.InputFileLocation interface. Use any of the following constructors:

InputEncryptedFileLocation Location of encrypted secret chat file. InputTakeoutFileLocation Used to download a JSON file that will contain all personal data related to features that do not hav InputDocumentFileLocation Document location (video, voice, audio, basically every type except photo) InputSecureFileLocation Location of encrypted telegram passport file. InputPhotoFileLocation Use this object to download a photo with upload. get File method InputPhotoLegacyFileLocation DEPRECATED legacy photo file location InputFileLocation DEPRECATED location of a photo InputStickerSetThumb Location of stickerset thumbnail (see files ) InputGroupCallStream Chunk of a livestream InputPeerPhotoFileLocation Location of profile photo of channel/group/supergroup/user
