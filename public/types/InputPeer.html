InputPeer

Abstract type representing one of 7 possible constructors.

Type

Available Constructors

In Gogram, this type is represented as tg.InputPeer interface. Use any of the following constructors:

InputPeerUserFromMessage Defines a min user that was seen in a certain message of a certain chat. InputPeerChat Defines a chat for further interaction. InputPeerSelf Defines the current user. InputPeerUser Defines a user for further interaction. InputPeerChannelFromMessage Defines a min channel that was seen in a certain message of a certain chat. InputPeerChannel Defines a channel for further interaction. InputPeerEmpty An empty constructor, no user or chat is defined.
