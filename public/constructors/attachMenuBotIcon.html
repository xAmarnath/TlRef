AttachMenuBotIconObj

Represents an attachment menu icon for bot mini apps

constructor
attachMenuBotIcon#b2a7386b flags:# name:string icon:Document colors:flags.0?Vector<AttachMenuBotIconColor> = AttachMenuBotIcon;

Parameters

Name Type Description
Name string One of the following values: note that animated icons must be played when the user clicks on the button, activating the bot mini app. default_static - Default attachment menu icon in SVG format placeholder_static - Default placeholder for opened Web Apps in SVG format ios_static - Attachment menu icon in SVG format for the official i OS app ios_animated - Animated attachment menu icon in TGS format for the official i OS app android_animated - Animated attachment menu icon in TGS format for the official Android app macos_animated - Animated attachment menu icon in TGS format for the official native Mac OS app ios_side_menu_static - Side menu icon in PNG format for the official i OS app android_side_menu_static - Side menu icon in SVG format for the official android app macos_side_menu_static - Side menu icon in PNG format for the official native Mac OS app
Icon Document The actual icon file.
Colors Vector<AttachMenuBotIconColor> Attachment menu icon colors.

Returns

AttachMenuBotIcon
