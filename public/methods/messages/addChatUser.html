Layer 224

MessagesAddChatUser

Adds a user to a chat and sends a service message on it.

method Users
messages.addChatUser#cbc6d107 chat_id:long user_id:InputUser fwd_limit:int = messages.InvitedUsers;

Parameters

Name Type Description
ChatId long Chat ID
UserId InputUser User ID to be added
FwdLimit int Number of last messages to be forwarded

Returns

messages.InvitedUsers

Gogram Example

// MessagesAddChatUser - positional arguments
result, err := client.MessagesAddChatUser(int64(1234567890), &tg.InputUserSelf{}, 42)
if err != nil {
    // handle error
}
// result is *tg.MessagesInvitedUsers
