ChannelsEditBanned

Ban/unban/kick a user in a supergroup/channel.

method Users Bots
channels.editBanned#96e6cd81 channel:InputChannel participant:InputPeer banned_rights:ChatBannedRights = Updates;

Parameters

Name Type Description
Channel InputChannel The supergroup/channel.
Participant InputPeer Participant to ban
BannedRights ChatBannedRights The banned rights

Returns

Updates

Gogram Example

// ChannelsEditBanned - positional arguments
result, err := client.ChannelsEditBanned(&tg.InputChannel{ChannelID: int64(1234567890), AccessHash: int64(5678901234567890)}, &tg.InputPeerUser{UserID: int64(777000)}, &tg.ChatBannedRights{})
if err != nil {
    // handle error
}
// result is *tg.Updates

Code Type Description

400 MSG_ID_INVALID Invalid message ID provided.
400 PARTICIPANT_ID_INVALID The specified participant ID is invalid.
400 PEER_ID_INVALID The provided peer id is invalid.
400 USER_ADMIN_INVALID You're not an admin.
400 USER_ID_INVALID The provided user ID is invalid.
